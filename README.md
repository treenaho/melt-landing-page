# melt — Drink the dream.

A luxury liquid-glass landing page concept for a fictional collagen milkshake brand.
Single-page interactive design exploring pearlescent glassmorphism, scroll-driven
storytelling, and beauty-meets-dessert brand language.

**Designed by [Katrina Diab](#) · 2026 · Portfolio piece**

---

## What it is

- Four-section landing page: hero → flavor portals → ritual → choose your dream
- Scroll-pinned flavor portals with color-morphing backgrounds
- Real product video, real UGC photography
- Editorial timeline, animated stat counters, FAQ accordion
- Fully responsive, reduced-motion safe

## What it's built with

- Vanilla HTML, CSS, JavaScript — no frameworks, no build step
- GSAP + ScrollTrigger (via CDN) for scroll choreography
- Google Fonts: Bebas Neue + Poppins

## Run locally

Just open `index.html` in any browser. That's it.

## Deploy

This repo is set up to deploy on Vercel with zero config. Just connect the repo
on [vercel.com/new](https://vercel.com/new) and it ships.
